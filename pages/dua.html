Dataset Usage Agreement

To use the dataset, participants are required to follow all data usage agreements of the source datasets. In particular, participants should not try to identify the individuals.

For each dataset participants are subject to the following DUA:

  • OASIS1, OASIS2: the OASIS data usage agreement
  • BraTS24: CC-BY-NC 4.0
  • MSD Braintumor: CC BY-SA 4.0
  • IXI: CC BY-SA 4.0

The following datasets were uploaded with no data usage agreement, and can be used for research purposes:

  • MGH Wild (Permision granted by author).
  • NKI, see this link

The following datasets were downloaded from OpenNeuro which has the permissive C0 license:

  • SOOP
  • NIMH
  • DLBS
  • IDEAS
  • ARC
  • MBSR
  • UCLA
  • QTAB
  • AOMIC ID1000
